# APS Revit Addins

A collection of Autodesk Platform Services (APS) add-ins for Revit.

## Overview

This repository contains Revit add-ins built with the Autodesk Platform Services SDK.

## Getting Started

### Prerequisites

- Autodesk Revit 2022 or later
- .NET Framework 4.8
- Autodesk Platform Services account

### Installation

1. Clone this repository
2. Open the solution in Visual Studio
3. Build the project
4. Copy the output `.dll` and `.addin` files to your Revit add-ins folder
